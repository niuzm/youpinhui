# 优品汇商城项目
## 项目介绍
本项目后台使用spring+springMVC+MyBatis,前台使用boostrap+angularJS,数据库使用MySql，分布式使用dubbox+zookeeper实现。
登录功能使用SpringSecurity实现， 使用Bcrypt算法进行加密。<br/>
注意：要运行此项目，必须先安装zookeeper，配置项目中相关配置文件的duoobx配置。install各个依赖项目后再分别run service和web项目。
<br/>

## 优品汇运营商后台
### 功能介绍
###### 商家管理
商家审核：可查看发起入驻申请商家的详细信息，操作其状态为 审核通过/审核未通过/关闭。
###### 商品管理
品牌管理：包括对品牌的增删改查，分页使用pageHelper实现。<br/>
规格管理：包括对规格以及规格选项的增删改查，分页使用pageHelper实现。 页面可以动态添加删除控制规格选项的数量。（绑定集合变量实现）<br/>
模板管理：包括对模板及扩展属性的增删改查，分页使用pageHelper实现。 页面可以动态添加删除控制扩展属性的数量。（绑定集合变量实现）<br/>
关联规格和关联品牌下拉框使用select2实现多选。<br/>
分类管理：包括对分类的增删改查，该页面使用三级分类，实现了三级联动的效果，若下级分类不为空则不能删除上级页面<br/>
商品审核: 包括对未审核商品的查看，审核状态，和删除等功能  <br/>
###### 广告管理：
广告类型管理:包括对广告的类型的增删改查功能。 <br/>
广告管理:包括对广告的增删改查功能。图片上传使用fastDFS实现。可以控制网站首页显示哪些广告。 <br/>
## 优品汇商家后台
### 功能介绍
注册：商家可进行注册发起入驻申请，待运营商审核后可进行登录。<br/>
###### 商品管理：
商品录入：富文本编辑器使用kindeditor实现。使用fastDFS分布式的文件服务器实现文件上传功能。<br/>
商品管理：包括增加商品、删除商品、提交审核，屏蔽、修改以及上架下架等功能。
## 优品汇首页
###功能介绍
网站首页：使用SpringDataRedis缓存广告等数据 加快页面的记载速度。<br/>
搜索页面：使用SpringDataSolr实现搜索的功能（分词，多关键字搜索等）。使用ik analyzer实现中文的分词效果。
