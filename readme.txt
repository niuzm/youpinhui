# 优品汇商城项目
## 项目介绍
本项目后台使用spring+springMVC+MyBatis,前台使用boostrap+angularJS,数据库使用MySql，分布式使用dubbox+zookeeper实现。
登录功能使用SpringSecurity实现， 使用Bcrypt加密。<br/>
注意：要运行此项目，必须先安装zookeeper，配置项目中相关配置文件的duoobx配置。install各个依赖项目后再分别run service和web项目。
<br/>

## 优品汇运营商后台
### 功能介绍
###### 商家管理
商家审核：可查看发起入驻申请商家的详细信息，操作其状态为 审核通过/审核未通过/关闭。
###### 商品管理
品牌管理：包括对品牌的增删改查，分页使用pageHelper实现。<br/>
规格管理：包括对规格以及规格选项的增删改查，分页使用pageHelper实现。 页面可以动态添加删除控制规格选项的数量。（绑定集合变量实现）<br/>
模板管理：包括对模板及扩展属性的增删改查，分页使用pageHelper实现。 页面可以动态添加删除控制扩展属性的数量。（绑定集合变量实现）<br/>
关联规格和关联品牌下拉框使用select2实现多选。<br/>



## 优品汇商家后台
### 功能介绍
注册：商家可进行注册发起入驻申请，待运营商审核后可进行登录。
